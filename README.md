## How to use (via Web Application)    
1. Install the dependencies
```
pip3 install -r requirements.txt
```
2. Run the app
```
python3 streamlit run app.py
```

3. Open in your local browser 
```
http://localhost:8501
```

## How to use (via CLI)

1. Open the `CLI`
2. Edit the file "config.yml"
3. Install the dependencies
```
pip3 install -r requirements.txt
```
4. Run the app
```
python3 main.py
```
